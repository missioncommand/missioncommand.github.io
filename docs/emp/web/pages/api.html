Extensible Map Platform (EMP) API supports both web and Android application development. This section highlights some of the primary API capabilities and has the web and android documentation in a tab view to toggle between them

Using the examples

The provided examples may be run by clicking the "Try It!" button on the bottom of any of the tests and may be reset after each by clicking the "Reset" button.

It is best to reset the examples between each test; however examples may be run in conjunction with each other by clicking on a desired example.
